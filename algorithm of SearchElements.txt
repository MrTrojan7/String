Рекурсивный алгоритм сравнения строки с подстрокой (посимвольно):
1) если строки лексикографически равны: истина
2) если длина оригинальной строки < длины сравниваемой строки: ложь
3) если длина оригинальной строки >= длины переданной строки:
	3.1)функция поиска элементов (сравниваемый text, index (default = 0)
		int result = 0
		->если индех == длина this->text - длина text + 1 
		(если сравнивание перешло границу первой строки, часть которой могла бы сравниться со второй строкой)
			возвращаем 0;
		->поиск первого совпадения элементов (this->text[i+index] == text[i])
			->если символы не идентичны: 
				рекурсивный вызов (text, ++index) и сравнение символов, начиная
				со следующего индекса массива this->text
			->если символы идентичны:
				++result и продолжение сравнения
			->если колличество найденных совпадений == длина text: 
				полное совпадение строки, return result
		если возвращается число, равное колличеству строк в сравниваемой строке: истина
	3.2) ложь
4) ложь по дефолту
